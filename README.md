# Desafio de Projeto: Git/GitHub 🚀

Repositório criado para consolidar os aprendizados no bootcamp da **DIO**.

## 🛠️ Comandos Praticados
* `git init`: Inicializar repositório.
* `git add .`: Preparar arquivos (Staging).
* `git commit -m ""`: Criar ponto na história.
* `git push origin main`: Enviar ao GitHub.

## 📝 Markdown Basics
Aprendi a estruturar documentos com:
1. **Listas**: Organização de tarefas.
2. **Tabelas**: Comparação de dados.
3. **Citações**: Destaque de informações.
